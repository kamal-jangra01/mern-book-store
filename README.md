# MERN Book Store 📚

A full stack MERN application that allows users to manage books with authentication and CRUD operations.

---

## 🚀 Live Demo

Frontend  
https://bookstore-app-bay.vercel.app

Backend API  
https://bookstore-backend-143h.onrender.com

---

## 🛠 Tech Stack

Frontend
- React
- Vite
- Tailwind CSS
- Axios

Backend
- Node.js
- Express.js
- JWT Authentication

Database
- MongoDB Atlas

Deployment
- Vercel (Frontend)
- Render (Backend)

---

## ✨ Features

- User Signup & Login
- JWT Authentication
- Add Books
- Update Books
- Delete Books
- Protected API Routes
